# JqueryInputmaskRails

## Installation

Add this line to your application's Gemfile:

    gem 'jquery_inputmask_rails'

And then execute:

    $ bundle

Add the following to your `app/assets/javascripts/application.js`:

    //= require jquery.inputmask
